# Prompt: Quality Assurance Review

Review the chapter draft against:
- source extraction notes
- synthesis matrix
- findings traceability table
- QA checklist in `qa/QA_CHECKLIST.md`

Check for:
- unsupported statements
- inconsistent numbers or names
- duplicated paragraphs
- weak transitions
- recommendations not grounded in findings
- vague statements that need evidence
- unresolved placeholders

Output:
A markdown review note saved as:
`qa/review_[chapter-name].md`

Where possible, propose exact edits.
