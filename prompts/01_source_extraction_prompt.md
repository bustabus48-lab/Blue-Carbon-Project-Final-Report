# Prompt: Source Report Extraction

Read the source report in `source_reports/` and complete the template in `templates/source_extraction_template.md`.

Instructions:
- Extract only what is supported by the report.
- Keep numeric findings exact.
- Note page references where possible.
- Identify what is useful for the main report versus annexes.
- Flag contradictions, gaps, weak evidence, and unresolved issues.
- Do not write full prose sections of the final report yet.

Output file:
`notes/extraction_[source-name].md`
